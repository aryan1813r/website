# Mohsinaa Ahmed - Professional Website

A premium, feminine-leadership website for Mohsinaa Ahmed — Energy Transformation Coach, Podcast Host, and Creator of Quantum Leaders Program.

## HTML / CSS / JS Version (Multi-Page)

Open `index.html` in your browser — no build step required.

**Pages:**
- **index.html** — Home (hero, pain, transformation, explore cards)
- **about.html** — About Mohsinaa (bio, values, philosophy)
- **program.html** — Quantum Leaders Program (outcomes, pillars)
- **podcast.html** — Self Discovery Podcast (episodes)
- **contact.html** — Work With Me & Contact form

**Assets:**
- **styles.css** — Design system (Cormorant Garamond + Source Sans 3)
- **script.js** — Mobile menu toggle

Works on any static host (Netlify, Vercel, GitHub Pages, etc.).

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.
